# feedback-service

REST Service to validate whether a given comment is objectionable or not.

# Technology Stack
This service is written in Java and on Spring boot

Sample input
```bash
{
  "comment":"<comment goes here>"
}
```
Sample Output
```bash
{
    "feedback": "Thank you for the response"
}
```
